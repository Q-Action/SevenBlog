Mac 使用小技巧

使用 Mac 中遇到的问题及解决方法记录

塞翁
2022-01-26
2 min

Mac 老萌新在使用过程中还是会遇到挺多技巧性的东西，收集起来方便使用

# iPhone 充电一直断开问题

# 问题描述

iPhone 连上 MAC 之后会不断地断开又连上，断开又连上

# 解决办法

  1. 打开终端
  2. 输入 sudo killall -STOP -c usbd 并回车
  3. 输入 密码
  4. 再次连接 iPhone

# 注意事项

重启 Mac 的 USB 驱动，对 Mac 没有什么影响，可放心大胆使用。其他情况下，USB无法使用的时候，也可以用这个命令试试，也许能解决困扰的问题。

# Mac 修改 Host 文件

打开终端，输入以下代码即可编辑 hosts 文件。

sudo vi /etc/hosts

hosts 文件路径为 /etc/hosts ,快捷键 CMD+SHIFT+G 后输入路径，可以直接到达 hosts 文件。

# Mac 打开应用提示“ XX 已损坏，无法打开...”解决办法

# 问题原因

Mac电脑启用了安全机制，默认只信任Mac App Store下载的软件以及拥有开发者 ID 签名的软件，但是同时也阻止了没有开发者签名的 “老实软件”

# 解决办法

  • macOS Mojave 10.14及以下系统

    1. 打开「终端.app」

    2. 输入以下命令并回车

      sudo spctl --master-disable

    3. 输入开机密码回车

  • macOS Catalina 10.15及以上系统

    1. 打开「终端.app」

    2. 输入以下命令并回车

      sudo xattr -rd com.apple.quarantine 空格 软件的路径

      软件路径格式如 /Applications/Sketch.app，可以直接将从应用程序中将 APPICON 拖入终端。

    3. 输入开机密码回车

# 显示隐藏 Mac 的桌面文件

使用 Mac 能不安装软件就最好不安装，终端一条命令能解决问题，就使用终端命令。

# 隐藏桌面

defaults write com.apple.finder CreateDesktop -bool false && \
killall Finder

# 显示桌面 (默认)

defaults write com.apple.finder CreateDesktop -bool true && \
killall Finder
