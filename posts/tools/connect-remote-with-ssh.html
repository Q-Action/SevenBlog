ssh 登陆&断开远程服务器连接

塞翁
2021-11-02
2 min

需要登陆远程服务器的场景比较少，以至于每次要连接到远程服务器，总是要谷歌一下方法。痛定思痛，网上搜刮并整理出来，备用。

# 登录远程服务器

# 密码登陆

# 通过输入需要登陆的用户名、服务器 IP、服务器端口进行连接
# enter 后会要求输入登陆密码
# 密码校验成功后即可完成登陆
$ ssh userName@monitorIP -p port

# sshKey 登陆

# 通过输入需要登陆的用户名、服务器 IP、服务器端口后校验 sshKey 后进行连接
# demo：
# ssh root@192.168.9.9 -p 23333 -i ~/.ssh/demo-rsa
$ ssh userName@monitorIP -p port -i sshKeyPath

TIP

sshKey 登陆方式需要先将远程服务器的公钥导入到设备中。

# 退出远程服务器

序号方法备注
1关闭终端简单粗暴
2输入logout正常退出
3输入exit正常退出
4快捷键 Ctrl+D正常退出

# 创建 SSHKEY

通过 SSHKEY 协议连接远端服务器，可以省去输入账户密码的操作，即安全又便捷还可以免去记忆各个平台的账号密码。既然好处这么明显，有啥理由不用呢！

# 创建 SSHKEY - MAC

  1. 打开终端.

  2. 执行以下命令，创建一个新的 SSH KEY.

    $ ssh-keygen -t rsa -b 4096 -C "your_email@example.com"

    成功后会返回以下信息.

    > Generating public/private rsa key pair.

  3. 设置保存地址.

    直接 ENTER 选择默认地址.

  4. 设置密码.

    如果有需要的情况下，可以设置密码.

# 查看 SSH KEY

  1. 打开终端.

  2. 执行查看命令.

    $ ls -al ~/.ssh

    执行后，如果文件存在，会列出来，默认情况下为以下文件

    id_rsa.pub
id_ecdsa.pub
id_ed25519.pub

# 添加到 GITHUB 等使用

  1. 打开终端.

  2. 复制 SSH KEY.

    $ pbcopy < ~/.ssh/id_rsa.pub

  3. 将 SSH KEY 黏贴到 GITHUB 的 SSH KEYS 设置页面中
